# 🏠 Прогнозирование стоимости жилья для агентства недвижимости
ML‑модель + веб‑сервис для автоматической оценки объектов недвижимости

## 📌 Контекст
Агентство недвижимости хочет автоматизировать оценку стоимости объектов жилья. Ручная оценка занимает много времени, зависит от опыта риелтора и может приводить к ошибкам в ценообразовании. Это снижает скорость работы и влияет на прибыль.

**Цель проекта** — создать модель машинного обучения, которая прогнозирует стоимость объекта недвижимости по его характеристикам, а также предоставить простой веб‑сервис для получения оценки.

В рамках проекта проведена **очистка сырых исходных данных**, преобразование и создание новых признаков, кодирование категориальных признаков.
В процессе моделирования созданы модели линейной регрессии, катбустинга, случайного леса и, в итоге, стэкинга случайного леса и катбустинга. 
Проведен сравнительный анализ метрик от каждой модели и , в результате, сохранен лучший( стэкинг) для представления его в простейшем вебсервисе.

## 🎯 Основные этопы проекта
- Очистить и подготовить данные для моделирования.

- Построить и сравнить несколько моделей регрессии.

- Выбрать лучшую модель по метрикам качества.

- Сохранить модель и интегрировать её в веб‑приложение.

Обеспечить удобный интерфейс для ввода параметров объекта и получения прогноза.
